# Insect-Catch-Game

How to Play:

1. Choose your target! 
Select between a butterfly, dragonfly, bee, or ladybug.

2. Start the clock:
As soon as you click the image the game begins.

3. Swat!
Use the fly swatter cursor to click on the insects as they randomly pop up!

4. Survive
The game gets harder as more insects spawn. Are you ready?

Features:

1. Custom Swatter Cursor: for maximum smashing power!
2. Screen shake: Every hit with an impactful screen-vibration effect on every catch.
3. Responsive UI: A "sliding screen" transition system for a seamless flow from menu to gameplay.
4. Annoyance Factor: A hidden message that triggers once you reach a score of 20.

How to Install:

1. Clone this repository: git clone https://github.com/your-username/bug-smasher.git
2. Navigate to the folder: cd bug-smasher
3. Open index.html in your favorite browser
